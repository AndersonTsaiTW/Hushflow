# Hushflow Client

Auto-generated Serverpod client library for the Hushflow application.

## Usage

Add to your Flutter app's `pubspec.yaml`:

```yaml
dependencies:
  hushflow_client:
    path: ../hushflow_client
```

Then import and use:

```dart
import 'package:hushflow_client/hushflow_client.dart';

final client = Client('http://localhost:8080/')
  ..connectivityMonitor = FlutterConnectivityMonitor();
```

## Regenerating

This package is auto-generated by Serverpod. To regenerate after model changes:

```bash
cd ../hushflow_server
serverpod generate
```

> ⚠️ **Do not modify files in this package manually** - changes will be overwritten on regeneration.
